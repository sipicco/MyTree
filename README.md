# MyTree
A repository to study data tree operations.

## Description
The aim is to create a versatile data structure for efficient storage and retrieval of hierarchical data.
It allows for easy insertion, deletion, and searching of nodes

## Features
- **Dynamic Sizing**: Automatically adjusts to accommodate varying amounts of data.
- **Optimal Search Algorithms**: Implements advanced searching techniques for quick lookups.
- **Modular Design**: Easily extensible to include additional functionalities.

## Roadmap of methods to implent

- [x] PostOrderTraversal
- [x] Find
